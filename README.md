# rust_canny_edge_camera

# Requirements
- Linux (to use v4l)
- [Rust](https://www.rust-lang.org)

Tested on 1.22.1.

# Setup
```
sudo apt install libv4l-dev
sudo apt install libgtk-3-dev
```

# Run
```
cargo run
```
